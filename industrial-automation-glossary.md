# 工业自动化常用英语术语词汇表
本文档总结了工业自动化领域（PLC、DCS、HMI、SCADA）中最高频的术语、缩写及中文对照。适合技术文档工程师、现场工程师及翻译人员查阅。

## 1. 控制系统核心概念
   
| 英文术语 | 缩写 | 中文释义 |
| :--- | :--- | :--- |
| Programmable Logic Controller	| PLC	| 可编程逻辑控制器 |
Distributed Control System	| DCS	| 分布式控制系统 |
Supervisory Control and Data Acquisition	| SCADA |	数据采集与监视控制系统 |
| Human Machine Interface	| HMI	| 人机界面 |
| Remote Terminal Unit	| RTU	| 远程终端单元 |
| Fieldbus	| - |	现场总线 |
| Open-loop Control |	-	| 开环控制 |
| Closed-loop Control |	-	| 闭环控制 |
| PID Control (Proportional-Integral-Derivative)	| PID |	比例-积分-微分控制 |
