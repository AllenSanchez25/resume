## 1. 概述

### 1.1 什么是 Obsidian？

Obsidian 是一款基于 **Markdown** 的本地笔记管理工具。它的核心理念是 **“双向链接”**，让笔记之间可以相互关联，形成知识网络。

### 1.2 适用场景

- 个人知识库搭建
- 学习笔记与复习
- 项目文档管理
- 日常记录与灵感收集

### 1.3 核心特点

| 特点 | 说明 |
| :--- | :--- |
| **本地存储** | 笔记文件保存在本地，不依赖云端，数据完全由用户掌控 |
| **Markdown 支持** | 所有笔记均为标准 Markdown 格式，可随时用其他编辑器打开 |
| **双向链接** | 通过 `[[笔记名称]]` 建立笔记之间的关联，自动生成关系图谱 |
| **插件生态** | 官方和社区提供丰富的插件，可按需扩展功能 |
| **跨平台** | 支持 Windows、macOS、Linux、iOS、Android |

## 2. 安装与初始化

### 2.1 系统要求

| 平台 | 最低版本要求 |
| :--- | :--- |
| Windows | Windows 10 及以上 |
| macOS | macOS 10.14 及以上 |
| Linux | 支持 AppImage / Snap / Flatpak |
| 移动端 | iOS 13.0 及以上 / Android 8.0 及以上 |

### 2.2 下载与安装

1. 访问 Obsidian 官网：https://obsidian.md
2. 根据操作系统选择对应版本下载
3. 运行安装程序，按默认选项完成安装

### 2.3 创建第一个知识库（Vault）

1. 启动 Obsidian，点击 **“创建”**
2. 选择本地文件夹位置，命名知识库（如 `MyNotes`）
3. 点击 **“创建”** 进入主界面

> **提示**：知识库（Vault）本质是一个普通文件夹，所有笔记都以 `.md` 文件形式保存在其中。

## 3. 界面导览

Obsidian 主界面由以下四个核心区域组成：

| 区域 | 名称 | 说明 |
| :--- | :--- | :--- |
| **左侧** | 文件列表 | 显示知识库中所有笔记和文件夹 |
| **中间** | 编辑区 | 笔记的编辑和预览区域 |
| **右侧** | 侧边栏 | 显示关系图谱、反向链接、大纲等信息 |
| **顶部** | 工具栏 | 包含搜索、切换视图、设置等功能 |

> **快捷键**：`Ctrl/Cmd + P` 可快速调出命令面板，搜索任意操作。

## 4. 核心操作

### 4.1 创建笔记

- **方法一**：点击文件列表顶部的 **“新建笔记”** 图标
- **方法二**：快捷键 `Ctrl/Cmd + N`
- **方法三**：右键文件夹 → 选择 **“新建笔记”**

### 4.2 编辑笔记

Obsidian 使用 Markdown 语法，常用格式如下：

| 效果 | 语法 |
| :--- | :--- |
| 标题 | `# 一级标题`、`## 二级标题` |
| 加粗 | `**加粗文字**` |
| 斜体 | `*斜体文字*` |
| 无序列表 | `- 项目内容` |
| 有序列表 | `1. 项目内容` |
| 链接 | `[显示文字](URL)` |
| 图片 | `![图片名称](图片路径)` |
| 代码块 | `` `代码内容` `` |

### 4.3 双向链接（核心功能）

双向链接是 Obsidian 最强大的功能，用于建立笔记之间的关联。

**语法**：

---

# **示例笔记**：
- 在笔记 A 中输入 `[[项目管理]]`
- 点击该链接，会直接跳转到名为 `项目管理` 的笔记
- 在笔记 A 的侧边栏，会自动显示所有指向它的笔记（反向链接）

> **提示**：即使目标笔记不存在，点击链接后也会自动创建该笔记。

### 4.4 查看关系图谱

- 点击界面右侧的 **“关系图谱”** 图标
- 图谱以节点形式展示所有笔记及其关联关系
- 可拖动、缩放、筛选查看

## 5. 常用功能配置

### 5.1 设置主题

1. 点击左下角 **齿轮图标** → **外观**
2. 在 **“主题”** 下拉菜单中选择浅色或深色
3. 也可从社区下载第三方主题

### 5.2 安装插件

1. 打开 **设置** → **第三方插件**
2. 关闭 **“安全模式”**
3. 点击 **“浏览”** 搜索插件，点击 **“安装”**
4. 安装后在插件列表启用即可生效

**推荐新手安装的插件：**

| 插件名称 | 功能说明 |
| :--- | :--- |
| **Calendar** | 日历视图，便于按日期整理日记 |
| **Excalidraw** | 手绘风格画板，绘制流程图和示意图 |
| **Dataview** | 将笔记数据以表格、列表等形式检索展示 |
| **Templater** | 笔记模板，快速生成固定格式内容 |

## 6. 备份与同步

### 6.1 本地备份

Obsidian 的所有笔记均为本地文件，直接备份知识库文件夹即可。

- Windows：`C:\Users\用户名\Documents\MyNotes`
- macOS：`/Users/用户名/Documents/MyNotes`

### 6.2 云同步方案

| 方案 | 适用场景 | 费用 |
| :--- | :--- | :--- |
| **Obsidian Sync** | 官方同步服务，全平台支持 | 付费 |
| **iCloud** | Apple 设备用户 | 免费（有存储限制） |
| **OneDrive / 坚果云** | Windows / 多端用户 | 免费 / 付费 |
| **Git** | 开发者，需要版本管理 | 免费 |

## 7. 常见问题

| 问题 | 解决方法 |
| :--- | :--- |
| **笔记无法打开** | 检查文件是否被移动或删除，确认路径正确 |
| **图片无法显示** | 图片需放在知识库文件夹内，使用相对路径引用 |
| **插件不生效** | 检查是否已启用插件，检查是否关闭安全模式 |
| **搜索不到笔记** | 确认搜索范围是否正确，检查是否有拼写错误 |
